# Android-News-App

这是大三下学期实训的时候，做的一个小型安卓新闻app

* 拥有如下功能：
  * 1、app首页3秒的欢迎界面
  * 2、偏好选择页面
  * 3、主页面
  * 4、设置页面
  * 5、浏览页面
  * 6、等待页面
